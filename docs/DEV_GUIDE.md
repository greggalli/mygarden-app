# 🧑‍💻 MyGarden App – Guide développeur

## Stack
- React + Vite
- CSS pur
- React Router
- localStorage

## Démarrage
npm install
npm run dev

## Arborescence
src/
 ├─ data/
 ├─ components/
 └─ pages/

## Conventions
- PascalCase pour composants
- camelCase pour fonctions
- Data JSON modifiable manuellement
