# 🎨 MyGarden App – Guide d’interface

## Navigation
- /zones → carte du jardin
- /zones/:id → vue d'une zone
- /species → liste d'espèces
- /plants → liste de plantations
- /add-plant → ajout d'espèce / plantation
- /admin → gestion JSON

## Design
- 2 colonnes : carte + liste
- Vert #2e7d32, tons clairs
- Layout flex, responsive

## Comportement
- Données locales (localStorage)
- SPA sans rechargement
