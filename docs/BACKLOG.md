# 🌿 MyGarden App – Backlog et Roadmap

## 🧱 1. Base existante

✅ Fonctionnalités déjà implémentées :
- Architecture React + routing
- Données persistées (localStorage + JSON)
- Ajout d’espèces et de plantations
- Navigation claire : Carte / Espèces / Plantations / Ajouter / Admin

## 🧭 2. Backlog à court terme

| Priorité | Fonctionnalité | Description |
|-----------|----------------|--------------|
| 🔥 | Édition / suppression d’espèce | Modifier ou supprimer une espèce existante |
| 🔥 | Édition / suppression de plantation | Modifier ou supprimer une plante réelle |
| ⚙️ | Export / Import JSON | Sauvegarde et restauration du jeu de données |
| ⚙️ | Gestion des tâches | Associer actions d’entretien aux plantes |
| 🌱 | Affichage photo | Afficher la photo d’espèce |
| 🌱 | Statistiques globales | Vue macro des plantes par zone ou espèce |

## 🧩 3. Backlog moyen terme

- Interaction directe avec la carte (drag & drop des pins)
- Synchronisation par fichier partagé (Drive / Git)
- Version mobile responsive
- Notifications ou calendrier d’entretien
- Intégration API PlantNet
- Statistiques avancées et rapports

## 🧰 4. Maintenance

- Nettoyage CSS
- Structuration data/ en JSON
- Couche d’import/export
- Tests unitaires helpers de tri/filtrage

## 🗓️ 5. Planification

| Sprint | Objectif principal | Contenu |
|--------|--------------------|----------|
| 1 | CRUD complet local | Édition + suppression |
| 2 | Sauvegarde / restauration | Export / import JSON |
| 3 | Interaction carte | Drag & drop pins |
| 4 | Gestion de l’entretien | Tâches planifiées |
| 5 | Vision macro | Statistiques |
